üìö About ParticleText.js

ParticleText.js is a lightweight, dependency-free JavaScript library that creates stunning animated text effects using particles. The library renders text as a collection of animated particles that explode when you move your mouse near them and smoothly animate back to form the original text shape.

Features: Multi-color support, responsive breakpoints, touch interaction, performance optimization, and accessibility support (respects prefers-reduced-motion).

üöÄ Getting Started

Since this package is unreleased (v0.1.0), include the source file directly in your HTML:

<!-- Add a canvas element --> <canvas id="canvas" width="1200" height="400" data-text="Hello World"></canvas> <!-- Include the library --> <script src="path/to/particleText.js"></script> <!-- Initialize --> <script> initParticleJS('#canvas', { colors: ['#695aa6', '#ABC121'] }); </script>
