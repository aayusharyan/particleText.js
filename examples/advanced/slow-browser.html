ParticleText.js can automatically detect slow browsers/devices and optimize performance by reducing quality. This example uses a lower threshold to demonstrate the feature.

üí° How It Works:

The library monitors frame render time. If it exceeds the threshold, it automatically:

  • Reduces canvas resolution
  • Decreases particle size
  • Adjusts friction for faster convergence
  • Optimizes explosion radius
