This example demonstrates programmatic control over the animation. With autoAnimate set to false, you can start, stop, and manually render single frames using the provided methods.

Animation Controls:

Animation Status:

Animation Running: false
Particle Count: 0

Configuration:

// Initialize with autoAnimate: false var particleText = initParticleJS('#canvas', { text: 'MANUAL CONTROL', colors: ['#695aa6', '#E74C3C', '#3498DB'], autoAnimate: false // Don't start automatically }); // Start animation programmatically particleText.startAnimation(); // Stop animation and cleanup particleText.destroy(); // Render a single frame (no animation loop) particleText.forceRequestAnimationFrame(); // Check animation status console.log(particleText.isAnimating); // true or false
