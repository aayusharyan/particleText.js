Understanding the Difference

The trackCursorOnlyInsideCanvas option gives you control over when particles should react to cursor movement.

Unrestricted Mode (Left/Top - Default)

trackCursorOnlyInsideCanvas: false // Default behavior

Particles track your cursor position globally. Even when your cursor is outside the canvas, particles calculate distance and react accordingly. This creates a smooth, fluid experience as particles begin responding before you even enter the canvas area.

Restricted Mode (Right/Bottom)

trackCursorOnlyInsideCanvas: true // Restricted tracking

Particles only track cursor when it's within canvas boundaries. When your cursor leaves the canvas, the mouse position is reset to (-9999, -9999), causing all particles to immediately return to their rest positions. This provides clear visual feedback about interaction zones.

Use Case Examples

  • Multiple Canvas Instances: Use restricted mode to prevent one canvas from affecting particles in another
  • Header/Footer Effects: Use unrestricted for seamless integration with page scrolling
  • Interactive Galleries: Use restricted to clearly define which element the user is interacting with
  • Performance Optimization: Restricted mode can reduce calculations when cursor is far from canvas
