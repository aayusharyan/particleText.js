This example demonstrates small text rendering with fine particle detail. The small font size (40px) combined with tiny particles creates a subtle, delicate effect perfect for secondary text or decorative elements.

Configuration:

initParticleJS('#canvas', { text: 'small', fontSize: 40, particleRadius: { 'xs': { base: 1, rand: 1 } // Tiny particles (1-2px) }, colors: ['#695aa6', '#ABC121'] });
