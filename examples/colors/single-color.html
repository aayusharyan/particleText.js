This example demonstrates using a single color for all particles. The coral red color (#FF6B6B) gives the text a unified, cohesive appearance. All particles are the same color, creating a clean and professional look.

Color Used:

Coral Red: #FF6B6B - A vibrant, warm red that stands out on dark backgrounds

Configuration:

initParticleJS('#canvas', { text: 'SINGLE COLOR', colors: ['#FF6B6B'], // Single color in array fontSize: 100 });

üí° Tip:

Even when using a single color, you still need to provide it as an array: ['#FF6B6B']. This maintains consistency with the multi-color configuration format. The color format must be a 6-digit hexadecimal color code (#RRGGBB).
