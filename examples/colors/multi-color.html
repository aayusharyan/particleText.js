This example demonstrates both equal and weighted color distribution. The first canvas shows equal distribution where all colors have the same probability. The second canvas uses weighted colors to create a dominant red theme with accent colors.

Configuration - Equal Distribution:

initParticleJS('#canvas1', { text: 'RAINBOW TEXT', colors: [ '#FF6B6B', // Coral Red '#4ECDC4', // Turquoise '#45B7D1', // Sky Blue '#FFA07A', // Light Salmon '#98D8C8' // Mint Green ], fontSize: 90 });

Configuration - Weighted Distribution:

initParticleJS('#canvas2', { text: 'WEIGHTED TEXT', colors: [ { color: '#FF6B6B', weight: 10 }, // 10x more red (dominant) { color: '#4ECDC4', weight: 2 }, // 2x turquoise (accent) { color: '#45B7D1', weight: 1 }, // 1x blue (subtle) { color: '#FFA07A', weight: 1 }, // 1x salmon (subtle) { color: '#98D8C8', weight: 1 } // 1x mint (subtle) ], fontSize: 90 });

üí° How It Works:

Equal Distribution: When colors are provided as strings, each particle has an equal chance of getting any color from the array.

Weighted Distribution: When colors are provided as objects with weights, you control the frequency. A weight of 10 means 10x more particles with that color compared to weight 1.

Weight Rules: Weights must be positive integers (1, 2, 3, etc.). No decimals, zero, or negative values. Invalid weights will show a browser alert.

Tip: Use weighted colors to emphasize brand colors while keeping accent colors for visual interest. Weights are relative - weight 10 vs weight 5 gives the same ratio as weight 2 vs weight 1.
