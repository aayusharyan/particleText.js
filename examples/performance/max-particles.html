üî¢ Max Particles

Control the maximum number of particles to balance visual quality with performance. More particles create denser text but require more processing power.

1. Low Count 500 particles

Fast performance but less dense - good for older devices

2. Medium Count 2000 particles

Balanced quality and performance - recommended for most cases

3. High Count 5000 particles

Dense detail but may impact performance on slower devices

Configurations:

// Low Particle Count initParticleJS('#canvas1', { text: 'LOW', maxParticles: 500, colors: ['#695aa6'] }); // Medium Particle Count initParticleJS('#canvas2', { text: 'MEDIUM', maxParticles: 2000, colors: ['#FF6B6B'] }); // High Particle Count initParticleJS('#canvas3', { text: 'HIGH', maxParticles: 5000, colors: ['#E74C3C'] });

üí° Choosing Particle Count:

  • 500-1000: Mobile devices, simple text, older browsers
  • 2000-3000: Desktop, balanced quality, most applications
  • 4000-5000: High-end devices, maximum detail, hero sections
  • Default: 5000 particles (can be reduced for performance)
