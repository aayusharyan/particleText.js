This is the simplest example of ParticleText.js with minimal configuration. The text "ParticleText.JS" is rendered as purple particles (#695aa6) that explode when you move your mouse near them and smoothly return to their original positions. Try hovering your mouse over the text below!

üéØ What You're Seeing:

  • Particle Animation: Each letter is formed by small circular particles
  • Mouse Interaction: Particles explode away from your cursor within ~150px radius
  • Smooth Return: After explosion, particles smoothly animate back to form the text
  • Touch Support: On mobile, touch to trigger particle explosion
  • Auto Animation: Animation starts automatically on page load

Configuration:

// HTML Canvas Element <canvas id="canvas" width="2000" height="400" data-text="ParticleText.JS"></canvas> // JavaScript Initialization initParticleJS('#canvas', { text: 'ParticleText.JS', colors: ['#695aa6'] // Single purple color }); // That's it! The animation starts automatically.

Default Behavior:

When you don't specify configuration options, ParticleText.js uses sensible defaults:

// Default configuration values: { autoAnimate: true, // Start animation automatically fontSize: canvas.height / 2, // 200px in this example fontWeight: 'bold', textAlign: 'center', particleRadius: { // Responsive particle sizes 'xs': { base: 2, rand: 1 }, 'lg': { base: 3, rand: 2 } }, explosionRadius: { // Responsive explosion zones 'xs': 20, 'lg': 75, 'xxl': 150 }, friction: { // Particle movement speed base: 0.9, rand: 0.05 }, maxParticles: 5000 // Maximum particle count }
