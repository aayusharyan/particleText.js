‚ö™ Particle Density

This example demonstrates how particle size affects visual density and appearance. Smaller particles create a denser, more detailed look, while larger particles create a looser, more artistic effect.

1. Dense Small Particles

Fine detail with tiny 1-1.5px particles - creates smooth, dense text

2. Medium Density

Balanced 2-3px particles - the standard recommended size

3. Sparse Large Particles

Large 4-7px particles - artistic, looser composition

Configurations:

// Dense Small Particles initParticleJS('#canvas1', { text: 'DENSE', particleRadius: { 'xs': { base: 1, rand: 0.5 } }, colors: ['#4ECDC4', '#45B7D1'] }); // Medium Density (Default-like) initParticleJS('#canvas2', { text: 'MEDIUM', particleRadius: { 'xs': { base: 2, rand: 1 } }, colors: ['#FF6B6B', '#FFA07A'] }); // Sparse Large Particles initParticleJS('#canvas3', { text: 'SPARSE', particleRadius: { 'xs': { base: 4, rand: 3 } }, colors: ['#9B59B6', '#8E44AD'] });

  • Sparse (4-7px): Artistic style, large display text, creative projects
