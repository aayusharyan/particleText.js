üí• Explosion Radius

The explosion radius determines how far from your mouse cursor particles will be repelled. A larger radius creates a bigger "bubble" of interaction, while a smaller radius requires precise mouse positioning. Try moving your mouse over each text to feel the difference!

1. Small Explosion 50px radius

Tight explosion - particles only react when mouse is very close

2. Medium Explosion 150px radius

Standard explosion - balanced interaction zone (default)

3. Large Explosion 300px radius

Wide explosion - particles react from far away

Configurations:

// Small Explosion (50px) initParticleJS('#canvas1', { text: 'SMALL', explosionRadius: { 'xs': 50, 'lg': 50 }, colors: ['#695aa6'] }); // Medium Explosion (150px) initParticleJS('#canvas2', { text: 'MEDIUM', explosionRadius: { 'xs': 150, 'lg': 150 }, colors: ['#FF6B6B'] }); // Large Explosion (300px) initParticleJS('#canvas3', { text: 'LARGE', explosionRadius: { 'xs': 300, 'lg': 300 }, colors: ['#E74C3C'] });

üí° Choosing Explosion Radius:

  • Small (30-70px): Precise interaction, mobile-friendly, subtle effect
  • Medium (100-200px): Balanced, works for most screens, easy to trigger
  • Large (250-400px): Dramatic effect, desktop displays, artistic projects
  • Tip: Use responsive breakpoints to adjust radius based on screen size
