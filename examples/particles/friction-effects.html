üåä Friction Effects

Friction controls how quickly particles return to their original positions after being exploded. High friction creates slow, smooth movement with minimal bounce. Low friction creates fast, energetic movement with visible overshoot and oscillation. Try exploding particles in each example to see the different animation characteristics!

1. High Friction Slow & Smooth

Friction: 0.95 - Particles move slowly and smoothly with minimal bounce

2. Medium Friction Balanced

Friction: 0.9 - Default balanced movement (recommended)

3. Low Friction Fast & Bouncy

Friction: 0.8 - Fast, energetic movement with visible bounce/overshoot

Configurations:

// High Friction (Slow, Smooth) initParticleJS('#canvas1', { text: 'SMOOTH', friction: { base: 0.95, rand: 0.03 }, colors: ['#3498DB', '#2980B9'] }); // Medium Friction (Balanced - Default) initParticleJS('#canvas2', { text: 'BALANCED', friction: { base: 0.9, rand: 0.05 }, colors: ['#95A5A6', '#7F8C8D'] }); // Low Friction (Fast, Bouncy) initParticleJS('#canvas3', { text: 'BOUNCY', friction: { base: 0.8, rand: 0.1 }, colors: ['#E74C3C', '#C0392B'] });

‚öôÔ∏è How Friction Works:

Friction values range from 0 to 1, where:

  • 0.95-0.98: Very high friction - slow, damped movement, no overshoot
  • 0.85-0.93: Moderate friction - natural, balanced feel (recommended)
  • 0.70-0.85: Low friction - fast, springy, energetic animation
  • rand: Adds variation between particles for organic feel

üí° Use Cases:

  • High Friction: Professional sites, calm aesthetic, subtle interactions
  • Medium Friction: General use, balanced animation, most applications
  • Low Friction: Playful sites, energetic brands, gaming, entertainment
